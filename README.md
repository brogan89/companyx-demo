# CompanyX-Demo
 
A small demo showcasing the use of Network For GameObjects as per [requirements](./Unity_Multiplayer_game_demo_proposal_(Netcode_for_GameObjects).pdf)


## Running Two Instances 

### Published Build

To run two instances of the built game:

- Double click on `CompanyX.exe` in Builds in folder, and again for second instance.


### In Editor

To run 2 instances of the game from the editor :

- Ppen the **Multiplayer Center Window** from `Window > Multiplay Center`
- Select `2` for **Number of Players per Session**
- Press `Ctrl + B` to build and run the game, or choose `File > Build And Run`
